Simple-Blockchain is a simplified blockchain project created to better understand and learn about blockchain technology. It serves as a basic implementation for educational purposes.

## Installation
To install the project locally, follow these steps:
1. Clone the repository: `git clone https://github.com/ErlisI/Simple-Blockchain.git`
2. Change to the project directory: `cd Simple-Blockchain`
3. Install dependencies: `npm install`
